## Instructions

1. Install Docker on your workstation (see instructions). Make sure you're on the latest version of docker-compose.

2. Run the following commands in your terminal:

```bash
git clone https://github.com/tegonhq/docker.git
git clone https://github.com/tegonhq/tegon.git
cd docker
```

3. Create a `.env` file

```bash
cp .env.example .env
```

4. Run the start script and follow the prompts, We handle 2 major steps in start script setting up trigger.dev and setting up tegon for you. 

```bash
./start.sh
```

5. Depending on the contribution (server or frontend) you can comment that service or change the port. 


## Server setup

1. Now go to server folder in the tegon folder

```bash
cd ../tegon/server
pnpm
```

2. Create a `.env` file inside server folder by copying the content from `.env.example` and change values according to your need.

```bash
pnpm migrate:dev
pnpm start:dev
```

## Frontend setup

1. Now you can move to frontend

```bash
cd ../webapp
pnpm 
pnpm dev
```
