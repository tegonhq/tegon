---
title: "Deploy Tegon"
description: "Deploying Tegon Open-Source just takes 1 minute."
---

1. Install Docker on your workstation (see instructions). Make sure you're on the latest version of docker-compose.

2. Run the following commands in your terminal:

```bash
git clone https://github.com/tegonhq/tegon.git
cd tegon
docker-compose up -d
```

3. Create user using the following commands

```bash
docker exec -ti tegon-server sh
cd scripts
node createUserWorkspaceTeam
```

4. You can now check tegon at `http://localhost:8000`
