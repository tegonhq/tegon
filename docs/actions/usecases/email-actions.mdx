## Overview

Whenever you forward an email to your team's assigned email address, this action will create a triage issue, including all email thread details in the description. 
To find your assigned email address, navigate to Settings -> Actions -> Email -> Config.

<Tip>You can find the whole code for this action [here](https://github.com/tegonhq/tegon/tree/main/actions/email)</Tip>
