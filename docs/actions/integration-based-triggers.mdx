Run a trigger when a integration sends a webhook request.

### Example config.json

```json
{
  "slug": "slack",
  "name": "Slack",
  "triggers": [
    {
      "type": "on_create",
      "entities": [ "IssueComment", "LinkedIssue" ]
    },
    {
      "type": "source_webhook",
      "entities": [ "slack" ]
    }
  ],
  "integrations": [ "slack" ]
}
```

<Tip>You can find the whole code for this example slack trigger [here](https://github.com/tegonhq/tegon/tree/main/actions/slack)</Tip>
